# Aria Agentworks

Open-source AI workflow systems for marketing, outreach, voice agents, and operator automation.

## Focus areas

- AI workflow orchestration for operator teams
- Outreach and sales automation infrastructure
- Voice-agent workflows for domain-specific operations
- Internal tools that combine dashboards, automations, and human review loops

## Featured repositories

- [ariaagent-solutions](https://github.com/aria-agentworks/ariaagent-solutions)  
  Open-source AI workflow automation stack for generating, deploying, and monitoring ad campaigns.
- [sales-intelligence-mcp](https://github.com/aria-agentworks/sales-intelligence-mcp)  
  MCP toolkit for company research, lead scoring, outreach automation, and CRM sync.
- [dentalvoice-ai](https://github.com/aria-agentworks/dentalvoice-ai)  
  White-label AI voice agent starter for dental office intake, scheduling, and patient communication workflows.
- [career-ops-hub](https://github.com/aria-agentworks/career-ops-hub)  
  AI job search workspace for resume tailoring, role tracking, and career operations workflows.
- [AI-Creative-Studio](https://github.com/aria-agentworks/AI-Creative-Studio)  
  AI image generation studio for NVIDIA NIM and Flux image workflows.

## Current flagship

[`ariaagent-solutions`](https://github.com/aria-agentworks/ariaagent-solutions) is the main public repository at the moment.

Recent improvements include:

- public roadmap and maintainer docs
- contribution and security guidance
- issue backlog for new contributors
- release history and validated lint/build flow

## Contributing

If you want to collaborate, start with the open issues in the flagship repo:

- setup and troubleshooting improvements
- service health checks and observability
- operator timeline and dashboard workflow improvements
